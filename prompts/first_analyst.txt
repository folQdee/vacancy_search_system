Извлеки навыки из следующего описания вакансий.

Для КАЖДОГО навыка:
- Отнеси его к одной из категорий:
    - languages
    - frameworks
    - infrastructure
    - soft_skills
- Добавь свойства:
    - "name": название навыка
    - "demand": одно из ["critical", "important", "nice-to-have"], где указана важность навыка
    - "trend": одно из ["growing", "stable", "declining"], где указан рыночный тренд навыка

Верни СТРОГО ТОЛЬКО JSON в следующем формате:

{
  "role": "{role}",
  "languages": [
    {
      "name": "Python",
      "demand": "critical",
      "trend": "growing"
    }
  ],
  "frameworks": [],
  "infrastructure": [],
  "soft_skills": []
}

Поле "role" должно быть строкой и совпадать с названием специальности: {role}

Не добавляй никакой текст вне JSON.
