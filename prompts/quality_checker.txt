Ты senior QA-аналитик, проверяющий карьерный аналитический отчёт.

Вход:
Объединенный JSON, содержащий:
- skill_map (навыки с demand и trend)
- salary_table (вилки зарплат по грейдам и регионам)
- learning_path (план из 3 фаз)
- gap_analysis
- portfolio_project

Твоя задача: оценить согласованность и качество отчёта.

Что нужно проверить:

1. Согласованность зарплат и навыков:
- Высокие зарплаты (Senior/Lead) должны соответствовать навыкам с "critical" и "growing"
- Если много навыков "nice-to-have" или "declining", но зарплаты высокие -> добавь warning

2. Согласованность learning_path:
- Навыки с "declining" не должны сильно приоритизироваться в learning_path
- Если это происходит -> добавь warning

3. Согласованность gap_analysis:
- "quick_wins" должны в основном включать "important" или "nice-to-have"
- "long_term" должны включать "critical" навыки

4. Согласованность portfolio_project:
- Проект должен использовать релевантные навыки с "critical" или "growing"
- Если нет -> добавь warning

Выход:

Верни СТРОГО ТОЛЬКО JSON:

{
  "quality_score": {
    "score": 0,
    "justification": "Короткое объяснение (1-3 предложения)"
  },
  "warnings": [
    "warning 1",
    "warning 2"
  ],
  "is_consistent": true
}

Правила:
- score: целое число от 0 до 100
- is_consistent = false, если есть существенные противоречия
- warnings может быть пустым, если противоречий нет

Входной JSON:
{full_report}
